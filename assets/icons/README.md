# 图标说明

这个目录包含了应用的图标文件。

## 图标要求

### macOS
- `icon.icns` - macOS应用图标
- 建议尺寸：1024x1024px

### Windows  
- `icon.ico` - Windows应用图标
- 建议尺寸：256x256px

### Linux
- `icon.png` - Linux应用图标  
- 建议尺寸：512x512px

## 当前图标

当前使用的是一个现代化的AI图像生成工具图标，包含：
- 渐变紫色背景
- 相机图标代表图像生成
- 金色星星代表AI功能
- "Fal.ai Studio"文字

如需更换图标，请替换对应格式的文件。
